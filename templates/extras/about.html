About this Project

When I started this project, the first idea was to make a project that will make the lives of businessmen easier by giving them insights on their sales. But as the project grew it has softly turned it into a personal expenses management app. Future scope of this project is to continue to add new features that help the user to properly focus on themselves and increase their productivity and profitability.

Introduction

This is a Summer Training Project. I have done my course from Coursera - 6 weeks starting from 22nd July to 2nd sept.

Course Certificate

Purpose

  1. To develop an easy to use web based interface where customers can easily store and monitor their expenses.
  2. Provide a visual representation of their expenditure.
  3. As many businesses are shifting to online platforms, everyone needs a reliable system to store and monitor their expenses, stocks or any other asset.
  4. Use user reviews and feedbacks to maintain and improve trust and loyalty
